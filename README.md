# 🔮 Astrology Prediction Website

## 📌 Project Overview
- A web-based astrology application that predicts future outcomes
- Generates horoscope and predictions based on user details
- Designed with a simple, attractive, and user-friendly interface
- Focuses on fun, curiosity, and basic astrological insights

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## ✨ Features
- User inputs personal details such as:
  - Name
  - Date of Birth
- Generates future predictions based on provided data
- Displays personality traits and general life insights
- Predicts:
  - Career direction
  - Love and relationships
  - Health outlook
  - Luck and daily fortune
- Instant result generation without page reload
- Simple and clean UI for better readability

---

## 🧠 Prediction Logic
- JavaScript processes user inputs
- Date of birth is used to determine zodiac sign
- Predefined prediction sets are mapped to zodiac signs
- Name is used to personalize the prediction output
- Randomized elements ensure slightly different predictions each time
